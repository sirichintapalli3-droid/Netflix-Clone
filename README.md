# Netflix-Clone

A simple Netflix clone built using HTML, CSS, and JavaScript.  
This project replicates the look and feel of Netflix’s homepage and showcases responsive design, interactive UI, and dynamic content rendering.

# Features
- Responsive Netflix-style homepage layout
- Movie/TV show sections with horizontal scrolling
- Hover effects and interactive UI
- Basic JavaScript functionality for dynamic content
- Clean and modern design using pure HTML, CSS, and JS

# Technologies Used
- HTML – Structure of the web pages
- CSS – Styling, layout, and responsiveness
- JavaScript– Interactivity and dynamic element
